Color Changing Boxes
Project Description

This is a simple web project that allows users to change the background color of the webpage by clicking on one of the colored boxes. Each box is associated with a specific color. When a box is clicked, the page background changes to that color.

The project demonstrates the use of HTML, CSS, and JavaScript, including DOM manipulation and event listeners.

Features

Interactive boxes: Click on any colored box to change the background.

Dynamic background color: The page background updates based on the clicked box.

Simple and scalable code: Uses querySelectorAll and forEach for easy event handling.

Clean design: Easy to understand for beginners learning DOM and JS events.

Technologies Used

HTML – for the structure of the webpage.

CSS – for styling the boxes and page layout.

JavaScript – for handling click events and changing the background dynamically.
